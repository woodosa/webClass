영역구성태그-div

영역을 나눌 때 사용하는 태그

컨텐츠의 맥락에 따라 영역을 나눔

영역구성태그는 자유롭게 사용할 수 있으나, 좌우로 뚜렷하게 나뉘어진 컨텐츠는 무조건 영역구성 태그를 사용하여 나눈다.

div로 부터 파생된 영역구성 태그들

영역의 쓰임새에 따라 태그명을 새롭게 만듦

css에서 나뉘어진 영역단위로 배치를 할 수 있기 때문에 좌우로 나뉘어진 영역은 반드시 영역태그로 구성되어야 한다.

  • header - 홈페이지의 머릿글 컨텐츠에 사요오디는 영역태그 (로고, 메뉴)
  • footer - 홈페이지의 하단 글 컨텐츠에 사용되는 영역태그 (회사 정보)
  • main - 홈페이지의 메인 컨텐츠를 전반적으로 다룰 때 사용 ( main 태그는 html 파일 당 1개만 사용 가능
  • section/article - 구획을 나눌 때 사용
  • nav - 홈페이지의 메뉴단을 만들 때 사용하는 영역태그
  • figure - 이미지나 영상의 전용 영역 태그
  • aside - 홈페이지의 독립된 사이드 컨텐츠를 만들 때 사용되는 영역태그 (다른태그의 공간 안에 넣지 않아도 된다.)
