함수 function

자바스크립트에서 동작을 표현하는 데이터 형태

function이라는 키워드를 통해 자바스크립트의 동작을 표현하는 함수를 만들 수 있다.

function 함수명(재료){동작할 소스코드}

  1. 함수는 정의를 먼저해야함
  2. 정의한 함수를 실행함

함수의 매개변수

함수는 매개변수(파라미터(재료))를 입력받을 수 있는 형태로 만들 수 있다.

함수를 실행하는 실행부에서 함수명(인자)형태로 함수를 실행할 수 있다.

매개변수의 개수는 제한이 없다.

함수 안에서 return

함수 안에서 return구문을 쓰게 되면 return 구문 오른쪽의 결과가 함수가 사용되었던 그 자리에 남기고 함수는 그 즉시 종료된다.
