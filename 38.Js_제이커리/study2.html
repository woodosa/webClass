태그선택

$("css선택자") $는 jquery함수의 약자로 괄호 안의 css선택자를 찾아서 함수가 사용된 자리에 제이쿼리 문서객체를 리턴한다.

사용되는 메서드

$(document).ready(function(){})

이 함수는 문서가 body태그 끝까지 읽혀진 뒤 한번 실행되는 예약함수로써 함수의 재료는 실행될 동작을 입력할 수 있는 함수

제이쿼리 문서객체.css()

css(함수는 제이쿼리 전용 메서드이며, 태그의 인라인 스타일로 스타일을 입력해주는 메서드이다.

attr("속성명","속성값")

네이버바로가기

attr()함수는 매개변수를 2개입력하면 선택한 속성의 속성값을 입력하는 용도로 사용되고, 매개변수를 1개만 싱별갛면 선택한 태그의 이볅한 속성에 저장된 속성값을 그리고 리턴하는 용도로 쓰인다.

text()/html()

Lorem ipsum dolor sit amet consectetur adipisicing elit. Adipisci praesentium inventore odio reprehenderit id iste voluptate? Officia at natus recusandae repudiandae consequuntur, repellendus dolorem dolorum, odit ipsum ratione quibusdam totam!

text()함수/html()함수는 선택한 태그의 내용을 새롭게 교체할 때 사용하는 함수이다. text()함수는 입력한 문자데이터를 그대로 선택한 태그의 내용으로 입력하며, html()함수는 입력한 문자데이터 중 태그문법을 포함하고 있다면 태그를 생성해서 입력한다.

text()함수는 매개변수를 입력하지 않았을 때, 선택한 태그가 갖고있는 택스트 내용을 리턴한다.

addClass{}/removeClass()

선택한 태그를 추가하거나 삭제할 수 있다.

포함관계를 이용한 태그를 선택할 때 사용하는 메서드(함수)

  • next()- 선택한 태그기준 다음형제 태그선택
  • prev()- 선택한 태그기준 이전형제 태그선택
  • children() -선택한 태그기준 자식태그 선택
  • parent()- 선택한 태그기준 부모태그 선택
