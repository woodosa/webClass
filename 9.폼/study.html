홈페이지를 사용하는 사용자가 내용을 입력할 수 있도록 미리 제공하는 태그

label 태그는 이름표 태그로 사용하며 사용자가 입력하는 입력박스에 대한 이름을 달아줄 때 사용한다

label태그와 input태그를 서로 연결하기 위해서는 input태그의 id값을 label태그의 for속성에 그대로 적용한다.

label태그가 디자인적으로 마음에 들지 않더라도 실제로 html상에서 label태그를 구성해야하며, css에서 안보이게 처리하는 방법을 사용하여 수정한다.

fieldset은 가두리양식, legend는 필드셋 안에서의 제목

input type속성이 radio인 경우 select태그와 같이 제한된 내용을 입력하게끔 유도할 때 사용하며, radio버튼의 name속성 값을 동일하게 하면 여러개의 radio버튼 중 1개만 선택되게 할 있다.

input type "text" 태그의 경우는 짧은 한 줄의 텍스트를 입력시킬 때 사용하며, textarea태그는 장문의 텍스트를 입력시킬때 빌셍힌디

사용되는 속성

  • value - 입력박스에 들어가는 실제 정보
  • disabled - 선택한 입력박스를 비활성
  • required - 선택한 입력박스의 실제정보를 반드시 입력시킬 때 사용
  • selected - 셀렉트 박스에서 선택한 option태그가 선택된 상태로 보이게 할 때 사용한다
  • checked - 체크박스/라디오 버튼이 체크된 상태로 표현할 때
