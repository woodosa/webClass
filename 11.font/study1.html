font

  • font-size -> 글씨의 크기를 지정하는 속성
  • font-weight -> 글씨의 굵기를 지정하는 속성
  • font-style -> 글씨의 스타일을 지정하는 속성
  • line-height -> 글씨의 행간을 조절하는 속성
  • font-family -> 글씨체를 변경할 때 사용하는 속성
  • text-indent -> 텍스트 첫 줄 들여쓰기
  • text-decoration -> 문단꾸밈
  • letter-spacing -> 자간 설정
  • word-spacing -> 단어와 단어사이의 간격 설정
  • word-break -> 단어 쪼개기 설정
  • white-space -> 텍스트를 강제로 한줄로 나열하게 설정(줄바꿈 되지 않도록)

축약형 표현

css의 각각의 개별속성을 한번에 표현하기 위한 방법 이 때, 대체적으로는 축약속성에 각각의 개별속성값을 띄어쓰기로 구분하여 입력한다. 때때로 입력한 수치가 혼동이 생길 것을 방지할 목적으로 "/"를 사용하여 입력하는 경우가 있다. 모든 축약형 표현이 순서를 명확히 따지지는 않으나, 순서를 따지는 축약형 표현도 있으니 참고하자.
