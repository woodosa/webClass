CSS

css는 html태그를 스타일링하는 style언어이다.

css방식

  • inline-style(element-style)

    태그를 직접 선택하여 시작태그에 style이라는 속성을 해서 스타일을 입력하는 방법.

  • 내브스타일 시트

    head태그안에 style태그를 작성하여 스타일을 입력하는 방법

  • 외부스타일 시트

    html파일 바깥에 css파일을 생성해서 스타일을 입력하는 방법

문법

css태그를 선택해서 스타일을 입력하는 언어. 그렇기 땜누에 선택하는 부분과 스타일이 입력되는 부분으로 나뉜다.

선택자{스타일속성:스타일속성값}

;(세미콜론)은 구분자로 사용된다
